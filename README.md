# PropertyScraper

## Installation

Add this line to your application's Gemfile:

    gem 'property_scraper'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install property_scraper

## Usage

    PropertyScraper.scrape('https://demeure.com/properties/le-petit-hopital')
